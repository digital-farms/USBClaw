PROJECT: MINICLAW USB

MiniClaw — это оффлайн AI ассистент, который работает с USB-флешки.

Основные характеристики:
- Работает без интернета
- Запускается в 1 клик
- Использует локальные модели (Gemma, Qwen)
- Может анализировать локальные файлы через RAG систему

Секретный код проекта: 7429-ALPHA
